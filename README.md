Sistema de Procesamiento de Órdenes de Pago

Proyecto académico desarrollado para la materia Algoritmos y Estructuras de Datos de la Universidad Tecnológica Nacional (UTN-FRC).

Descripción

Aplicación desarrollada en Python para procesar órdenes de pago a partir de archivos de datos, aplicando distintos algoritmos de cálculo, búsqueda y análisis.

Tecnologías utilizadas

* Python
* Programación Orientada a Objetos (POO)
* Manejo de archivos CSV
* Archivos binarios (Pickle)
* Búsqueda binaria
* Matrices y vectores
* Procesamiento de datos
